C. web-reviewer 검증 체크리스트

5개 검증 카테고리

  • Accessibility — WCAG 2.2 AA, 시맨틱 HTML, 키보드, 색상 대비
  • Performance — FCP<1.8s, LCP<2.5s, TBT<200ms, CLS<0.1
  • SEO — meta/title, JSON-LD, heading 구조, OG tags
  • Design Quality — 계층구조, 타이포, 반응형, 인터랙션 상태
  • AI Slop Detection — 9개 AI 패턴 탐지 (보라 그래디언트, 3열 그리드 등)
